# **App Name**: Bengali Snaps

## Core Features:

- Photo Gallery: Showcase wedding photos categorized by events (Aiburobhat, wedding, reception).
- Photographer Profiles: Display profiles of photographers with their bios and specializations.
- Package Display: Showcase photography packages with details and pricing.
- Contact Form: Enable users to submit inquiries and booking requests.
- Style Suggestions: AI tool suggesting best angles, styles and captions, for new images uploaded to the gallery.

## Style Guidelines:

- Primary color: Gold (#FFD700) to symbolize the richness and tradition of Bengali weddings.
- Background color: Soft beige (#F5F5DC), providing a gentle contrast.
- Accent color: Deep red (#A62A2A) to highlight important elements.
- Headline font: 'Poppins', sans-serif, providing a modern clean and fashionable aesthetic.
- Body font: 'Noto Serif Bengali', serif, for cultural and elegant text.
- Use cultural motifs, such as shehnai and shankha, to represent Bengali culture.
- Clean and spacious layout to exhibit photographs beautifully. Utilize grids for photo galleries.
- Use smooth transitions and subtle hover effects for enhanced user experience.
